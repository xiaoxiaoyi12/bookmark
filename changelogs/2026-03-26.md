# 2026-03-26 变更记录

## 新增功能

- **书库页面** — 拖拽/点击导入 EPUB 和 PDF，封面网格展示，删除书籍
- **EPUB 阅读器** — epub.js 渲染，连续滚动模式，目录导航，左右键切换章节
- **PDF 阅读器** — pdfjs-dist 连续滚动架构，IntersectionObserver 懒渲染可见页
- **PDF 缩放** — 工具栏 +/-/重置按钮，Cmd/Ctrl±0 快捷键，范围 50%~400%
- **PDF 文字层** — TextLayer 精确覆盖 canvas，支持文字选中和复制，HiDPI 适配
- **文字高亮** — EPUB 和 PDF 均支持选中文字后弹出工具栏，5 种颜色高亮
- **引用到笔记** — 选中文字一键高亮并以引用块插入 Tiptap 编辑器
- **笔记面板** — Tiptap 富文本编辑器（H2/粗体/斜体/列表/引用/代码块），自动保存
- **高亮列表** — 笔记面板下方展示当前书籍所有高亮
- **阅读进度** — EPUB 和 PDF 均自动保存/恢复阅读位置
- **面板布局** — 目录/笔记可折叠，笔记面板宽度可拖拽调整
- **暗色主题** — 全局暗色背景，EPUB 内容通过 themes API 注入暗色样式

## Bug 修复

- PDF 文字层精确对齐 canvas，解决选中位置偏移问题
- PDF 高亮精确到选中文字而非整行（splitText 拆分文字节点）
- PDF 从单页切换改为连续滚动，解决滚动跳跃感
