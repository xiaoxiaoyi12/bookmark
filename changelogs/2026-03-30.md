# 2026-03-30 变更记录

## 新增功能

- **网页剪藏阅读** — 输入网页 URL，自动提取正文内容保存到书库
  - 使用 @mozilla/readability 提取网页正文（同 Firefox 阅读模式）
  - 通过 CORS 代理（allorigins）获取网页内容，无需后端
  - 自动提取标题、作者、og:image 封面
  - 修正网页中的相对路径（图片、链接）
- **网页阅读器** — 新增 `web` 格式阅读器，与 EPUB/PDF 阅读器并列
  - 干净的正文排版（max-width、行高、字体适配）
  - 自动生成目录（从 h1-h6 标题提取）
  - 文字选中、高亮标注、写笔记（复用现有体系）
  - 点击已高亮文字可取消高亮
  - 全文搜索支持
  - 阅读进度自动保存/恢复（滚动位置百分比）
  - 暗色主题完整适配
  - 显示原始网页 URL，可点击跳转
- **书库 URL 导入** — 顶栏新增「添加网页」按钮
  - 点击展开 URL 输入框，支持 Enter 提交、Esc 取消
  - 抓取中显示加载状态，失败显示错误提示
- **备份兼容** — 导出/导入备份支持 `web` 格式书籍和 `url` 字段
- **AI 助手** — 阅读器侧边栏集成 AI 对话功能
  - 支持 OpenAI 兼容 API（OpenAI / DeepSeek / 通义千问等，可自由切换）
  - 一键生成文档摘要
  - 基于文档内容自由问答
  - 选中文本一键翻译（所有阅读器格式均支持）
  - 流式输出（Streaming），实时显示 AI 回复
  - API 配置持久化存储（localStorage），支持预设快捷切换
  - 未配置 API Key 时友好提示引导设置
  - AI 面板独立开关，支持拖拽调整宽度
  - 暗色主题完整适配
